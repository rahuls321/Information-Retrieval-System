W3Schools is a freemium educational website for learning coding online. Developed in 1998, it derives its name from the World Wide Web but is not affiliated with the W3C.[1][2] W3Schools offers courses covering all aspects of web development.[3] It is run by Refsnes Data in Norway.[4][5]
